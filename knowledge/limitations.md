# Limitaciones y Consideraciones

## No hacemos
- Desarrollo de software personalizado desde cero (solo automatización de procesos existentes)
- Consultoría estratégica profunda (aunque identificamos oportunidades)
- Transformación digital completa (nos enfocamos en puntos de dolor específicos)

## Páginas web - Contexto importante
Las páginas web no son nuestro servicio principal, pero es importante entender: **si no generas ventas online, tu página web actual no funciona**. Las páginas que hacemos están diseñadas para transformar y vender, adaptadas a tus necesidades específicas.

## Consideraciones
- Requiere documentación clara de procesos actuales
- Algunos procesos pueden necesitar redefinición para automatizar bien
- La ROI depende de la complejidad del proceso y el tiempo ahorrado realista
